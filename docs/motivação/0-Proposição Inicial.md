Qual seria um bom grupo de pessoas para explorar a ideia de iniciar um negócio que gere uma renda extra? Não precisa ser alta, mas constante e escalável, com automação que possa ser criada e hospedada na internet. O que eles fariam?


a) Documentos de Planejamento e Requisitos:
    Especificação de Requisitos de Software (SRS - Software Requirements Specification): Detalha os requisitos funcionais

b) Plano de Negócios e Estratégia Técnica: Define o modelo de negócios

c) Análise de Riscos e Conformidade: Identifica riscos (ex.: fraudes, volatilidade) e planos para conformidade regulatória, incluindo requisitos técnicos para verificação de usuários.

d) Documentos de Design e Arquitetura

e) Documento de Arquitetura do Sistema (SAD - System Architecture Document): Descreve a estrutura geral, como microservices para escalabilidade, componentes principais  e fluxos de dados. 

f) Design de Banco de Dados: Esquema de banco (ex.: PostgreSQL ou MongoDB), incluindo ER Diagrams

g) Design de API: Especificações para APIs internas e externas (ex.: RESTful ou WebSocket para dados em tempo real
